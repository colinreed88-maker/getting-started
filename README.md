# Getting Started — AI at Flow

Welcome to the onboarding kit for AI-assisted development at Flow. This guide takes you from zero to building and deploying Flow-branded digital experiences, with no prior coding experience required.

## What Is in the Config Folder

The `config/` folder contains ready-to-install files for your AI tools:

- **cursor-rules/** — 7 rule files that teach the AI Flow's brand, conventions, and workflow
- **cursor-agents/** — 2 specialized AI agents (project-launcher, deck-builder)
- **cursor-skills/** — 1 interactive workflow (PRD generation)
- **claude-project-instructions.md** — Instructions for Claude Desktop
- **CLAUDE.md** — Memory file for Claude Code CLI
- **assets/fonts/** — Flow brand fonts (Roughwell, Generation 1970, Poppins)
- **assets/logos/** — Flow logo variants (SVG)
- **brand-guide.html** — Visual brand reference with color swatches, font samples, and Tailwind config

Setup guides 03 and 04 walk you through installing each of these.

## Prerequisites

Before starting, you will need:

- A Windows, Mac, or Linux computer
- Admin permissions to install software
- A GitHub account (free)
- A Cursor license (check with your manager)
- A Claude account (check with your manager)
