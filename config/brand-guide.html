Next.js / Tailwind Setup

Copy this into your project to get Flow brand tokens working with Tailwind CSS v4.

globals.css (CSS Custom Properties + Tailwind Theme)

/* Import Tailwind */ @import "tailwindcss"; /* Tailwind v4 theme — maps CSS vars to Tailwind tokens */ @theme inline { --font-sans: var(--font-poppins), "Poppins", ui-sans-serif, system-ui, sans-serif; --radius: 10px; --color-background: var(--bg); --color-foreground: var(--midnight); --color-card: #ffffff; --color-card-foreground: var(--midnight); --color-primary: var(--sunlight); --color-primary-foreground: #ffffff; --color-secondary: var(--moonlight); --color-secondary-foreground: var(--midnight); --color-muted: var(--moonlight); --color-muted-foreground: #888888; --color-accent: var(--moonlight); --color-accent-foreground: var(--midnight); --color-border: rgba(61, 61, 61, 0.08); --color-input: rgba(61, 61, 61, 0.12); --color-ring: var(--sunlight); } /* Flow brand custom properties */ :root { --moonlight: #F3EDDF; --midnight: #3D3D3D; --sunlight: #E89700; --heart: #F99AA9; --ocean: #7DADBB; --olive: #767317; --roots: #8C4500; --bg: #EDE8DA; } /* @font-face declarations — copy font files to public/fonts/ */ @font-face { font-family: "Roughwell"; src: url("/fonts/Roughwell.woff2") format("woff2"), url("/fonts/Roughwell.otf") format("opentype"); font-display: swap; } @font-face { font-family: "Generation1970"; src: url("/fonts/Generation1970-Light.woff2") format("woff2"), url("/fonts/Generation1970-Light.otf") format("opentype"); font-weight: 300; font-display: swap; } /* Body defaults */ body { font-family: "Poppins", sans-serif; background: var(--bg); color: var(--midnight); }

Design Principles

Flow IS: Welcoming, Elevated, Authentic, Calm, Soulful

Flow is NOT: Pretentious, Cheesy, Overbearing

Clean, elevated, timeless. Warm cream backgrounds, white cards with thin borders, soft shadows. No generic "tech blue." No pure black text. Mobile-first on everything except slide decks.
